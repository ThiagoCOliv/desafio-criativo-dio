# desafio-criativo-dio
Repositório para colocar os prompts de desafio criativo na DIO

# Prompts
````
Gere uma tabela de lista de compras para pessoas de baixa renda, com o objetivo de checar os itens do carrinho durante a compra. Considere o seguinte contexto: Compra do mês. O conteúdo deve ter Item, prioridade, marca preferencial, preço estimado. Evite itens de consumo rápido como carnes e hortifruti.
````

````
Gere um cronograma de estudos para profissionais da área da tecnologia, com o objetivo de aprender sobre inteligência artificial. Considere o seguinte contexto: o objetivo é estar atualizado com a ferramenta em no máximo dois meses, não há orçamento para cursos e o conhecimento prévio é básico. O conteúdo deve ter formato de tabela com conteúdo, dias e fontes para referência. Evite conteúdos muito introdutórios e foque em conhecimento prático.
````

````
Gere uma letra de música para uma banda gospel, com o objetivo de louvar a Deus. Considere o seguinte contexto: pop rock. O conteúdo deve ter formato de poema. Evite referências que não sejam da Igreja Católica.
````
